# MonkeyTraders - 실시간 금융 데이터 시각화 플랫폼

**현대적인 기술 스택을 기반으로 구축된 고성능 실시간 금융 데이터 시각화 웹
애플리케이션입니다.**

이 프로젝트는 복잡한 금융 데이터를 사용자 친화적인 인터페이스로 제공하는 것을
목표로 합니다. 확장성을 고려한 모노레포 아키텍처 위에서 Next.js의 서버 사이드
렌더링과 Tanstack Query의 강력한 데이터 관리 기능을 결합하여 뛰어난 성능과
안정적인 사용자 경험을 제공합니다.

## 🚀 주요 기능 (Key Features)

- **📈 실시간 자산 데이터 시각화**: 웹소켓 또는 주기적인 폴링을 통해 자산의
  현재가를 실시간으로 추적하고 동적으로 업데이트합니다.
- **📊 인터랙티브 캔들 차트**: TradingView(Lightweight Charts) 라이브러리를
  활용하여 사용자가 기간(Timeframe)을 변경하며 시세 추이를 분석할 수 있는
  인터랙티브 차트를 구현했습니다.
- **📜 무한 스크롤**: 차트 및 일별 시세 테이블에 무한 스크롤을 적용하여,
  사용자가 과거 데이터를 끊김 없이 탐색할 수 있도록 최적의 사용자 경험을
  제공합니다.
- **🌐 반응형 UI**: TailwindCSS를 기반으로 모바일, 태블릿, 데스크톱 등 모든
  디바이스에서 최적화된 UI/UX를 제공합니다.
- **🎨 유연한 컴포넌트 기반 설계**: `shadcn/ui`를 기반으로 구축된 재사용 가능한
  컴포넌트 시스템을 통해 일관성 있고 유지보수하기 쉬운 UI를 구현했습니다.

## 🏛️ 아키텍처 및 기술적 결정 (Architectural Highlights)

이 프로젝트는 단순한 기능 구현을 넘어, 대규모 애플리케이션으로의 성장을 염두에
둔 깊이 있는 아키텍처 설계를 적용했습니다.

### 1. 모노레포 (pnpm Workspaces)

- **목적**: 코드 재사용성을 극대화하고 패키지 간의 의존성을 명확하게 관리하기
  위해 모노레포 구조를 채택했습니다.
- **구조**:
  - `apps/web`: 사용자와 직접 상호작용하는 Next.js 프론트엔드
    애플리케이션입니다.
  - `packages/*`: 도메인별로 분리된 재사용 가능한 라이브러리들입니다.
    - `finance-core`: 금융 데이터 타입, 비즈니스 로직 등 핵심 로직을 포함합니다.
    - `finance-ui`: 차트, 테이블 등 금융 데이터 시각화에 특화된 UI 컴포넌트를
      관리합니다.
    - `ui`: 버튼, 탭 등 범용적으로 사용되는 기본 UI 컴포넌트 집합입니다.
    - `hooks`: 데이터 fetching, 상태 관리 등 커스텀 훅을 모아 재사용성을
      높입니다.

### 2. 상태 관리 전략: Tanstack Query + Zustand

- **Tanstack Query (React Query)**: 서버 상태(Server State) 관리를 전담합니다.
  API 요청, 캐싱, 동기화, 로딩/에러 상태 처리를 자동화하여 비동기 로직을
  선언적으로 관리합니다. **무한 스롤** 기능은 `useInfiniteQuery`를 활용하여
  효율적으로 구현했습니다.
- **Zustand**: 클라이언트 상태(Client State) 중 전역적으로 필요한 상태(예: 테마,
  통화 설정)를 최소한의 보일러플레이트로 간편하게 관리합니다.

### 3. 데이터 시각화: TradingView 차트와 무한 스크롤

- **TradingView(Lightweight Charts)**: 금융 차트 구현을 위해 성능이 검증된
  TradingView 라이브러리를 선택했습니다. 사용자가 선택한 `Timeframe`에 따라
  `useInfiniteQuery`가 새로운 데이터를 비동기적으로 불러와 차트에 동적으로
  추가하는 방식으로 **인터랙티브 캔들 차트**를 구현했습니다.
- **무한 스크롤**: 사용자가 차트의 과거 영역으로 스크롤하거나 테이블의 끝에
  도달했을 때, `useInfiniteQuery`의 `fetchNextPage` 함수를 트리거하여 다음
  페이지의 데이터를 가져옵니다. 이를 통해 초기 로딩 시간을 단축하고 사용자
  경험을 극대화했습니다.
